# Online Quiz Application

A premium graphical Online Quiz application built in Python utilizing CustomTkinter for beautiful modern styling and SQLite for persistent database handling.

## Features
- **Premium UI:** Powered by CustomTkinter for dark-mode, rounded corners, and fluid interactions.
- **Admin Dashboard:** Secure interface (`password: admin`) to add/delete multiple choice questions and review all past student scores cleanly segregated via a modern Tabview.
- **Student Exam Panel:** Take quizzes with a live countdown timer and a responsive completion progress bar.
- **Dynamic Grading:** Visually clear results page that evaluates the total score against an accuracy ratio circle.

## Setup Requirements

1. Make sure Python 3.8+ is installed.
2. Install the necessary dependencies:
```bash
pip install customtkinter
```

## Running the App

To launch the Premium UI application, run the core `app.py` script:
```bash
python app.py
```
