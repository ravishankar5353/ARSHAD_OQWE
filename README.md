# Knowledge Nexus - Online Quiz Platform

A premium Flask Web Application built with Python, HTML, CSS, and JS, featuring a sophisticated Glassmorphism UI, interactive Chart.js analytics, and circular timers.

## Features
- **Web Interface:** Fully responsive UI utilizing CSS Glassmorphism, neon gradient accents, and micro-interactions.
- **Admin Dashboard:** Secure interface (`admin/admin`) to add/delete questions and review globally aggregated analytics via interactive Chart.js pie graphs.
- **User Dashboard:** Track individual academic progression on a timeline layout (`user/password`).
- **Student Exam Panel:** Take quizzes with a live precision circular JS countdown timer and clickable option tiles.
- **Dynamic Grading:** Visually clear results page that evaluates the total score, shows elapsed time, and delivers a large color-coded letter grade.

## Setup Requirements

1. Ensure Python 3.8+ is installed.
2. Install the necessary dependencies:
```bash
pip install flask
```

## Running the Web Portal

To launch the web server, run the core `app.py` script:
```bash
python app.py
```
Then navigate to `http://127.0.0.1:5000` in your browser.
